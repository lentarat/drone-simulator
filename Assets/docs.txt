You can use SmartPropoPlus program to set up RC transmitters to emulate Gamepad Input. https://discussions.unity.com/t/using-a-radio-control-system-for-game-input/216735

Ідеї:
мапи:
	полігон для тренування польотами фпв дроном;
	полігон з стаціонарними та рухаючими цілями (мітками); 

види квадрокоптерів:
	камікадзе, бомбер з вибухівкою, дрон з дробовиком;

майстерня:
	можливість зміни компонентів дрона та його бойової частини;

режими гри:
	статичні або динамічні мітки (рухаються по мапі по заданих траєкторіях (траєкторій декілька)).
		3 рівні складності: 1 рівень спавнить наступну ціль лише після знищення попередньої, 2 рівень не враховує стан попередньої (продовжує спавнити), 3 рівень спавнить ще швидше;
	ворожі дрони, які необхідно збити собою або дробовиком;
